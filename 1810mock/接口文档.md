### 首页轮播图

- 请求地址: /banner
- 请求方式: get
- 参数说明: 无
  -----
- 返回值:
  | 值 | 类型 | 说明 | 创建人/修改人 | 修改时间 |
  | ------ | ------ | -------- | ------------- | -------- |
  | code | number | 200 成功 | ry/ry | 2019/9/9 |
  | banner | array |图片数组集合 | ry/ry | 2019/9/9 |
- banner 里包含的数据(单数据)
  | 值 | 类型 | 说明 | 创建人/修改人 | 修改时间 |
  | ------ | ------ | -------- | ------------- | -------- |
  | id | number | 图片 id | ry/ry | 2019/9/9 |
  | id | imgurl | 图片地址 | ry/ry | 2019/9/9 |
